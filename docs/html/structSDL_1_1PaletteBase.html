SDL::PaletteBase< T >::PaletteBase ( int  ncolors)
inline

Create a palette structure with the specified number of color entries.

The palette entries are initialized to white.

Parameters
ncolorsrepresents the number of color entries in the color palette.
Postcondition
this represents a new Palette structure convertible to true on success or converts to false on failure (e.g. if there wasn't enough memory); call GetError() for more information.
Thread safety:
It is safe to call this function from any thread.
Since
This function is available since SDL 3.2.0.
