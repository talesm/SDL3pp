Detailed Description

template<class Result, class... Args>
struct SDL::CallbackWrapper< std::function< Result(Args...)> >
Template Parameters
Fthe function type.

For the simpler case, where no transformation is done on the parameters, you can just pass CallOnce() or CallOnceSuffixed(). Otherwise use release().

In all cases, use Wrap to change the callback into a void* pointer.
